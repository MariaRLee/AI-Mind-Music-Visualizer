# AI-Mind Music Visualizer V2

白底、乾淨版腦波音樂視覺化網頁。

## V2 新增

- 白色、清爽、較現代的版面
- 新增「彩色冰淇淋」動畫模式
- 新增瀏覽器即時合成的倫巴與恰恰節奏
- 可不選音樂，只播放合成節奏並觀看動畫
- 可將合成節奏與本機腦波音樂一起播放
- 背景節奏音量可調整

## 隱私與版權

- 本機音樂只在瀏覽器中處理
- 不上傳、不儲存、不追蹤
- 不含外部音樂服務或第三方程式
- 倫巴、恰恰不是下載的歌曲，也不是任何現有錄音
- 節奏由 Web Audio API 即時合成，因此不包含商業錄音檔
- CSP 使用 `connect-src 'none'`，阻止網頁程式對外連線

## GitHub Pages

將以下檔案上傳到 repository 根目錄：

- `index.html`
- `style.css`
- `visualizer.js`

不要將私人、客戶專屬或尚未公開的腦波音樂放進 Public repository。
