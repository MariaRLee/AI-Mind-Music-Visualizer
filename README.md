# AI-Mind Music Visualizer V4.1

## V4.1 updates

- General music can use the basic visualizer.
- Maximum file size remains 100 MB.
- No hard duration limit.
- For smoother mobile use, music of 30 minutes or less is recommended.
- Added the bilingual `使用指南｜User Guide` button.
- Added an attractive bilingual introduction and operating guide.
- Added Hylove Business Inc. copyright.
- Added compatibility notices for Apple, Android, and LINE in-app browsers.
- Preserved all V4.0 visual modes, custom themes, percussion, tempo, intensity,
  bell accent, privacy, and local processing.
